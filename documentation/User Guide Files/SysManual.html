Where to Poop System Manual

Minimum hardware and software requirements.

This product has been tested on Google Chrome version 74 and Firefox version 66, however it can theoretically be run using any other browser as long as it supports JavaScript. Due to the limited testing this product has undergone, please alert the developers if it does not work on your browser or device so we can improve the product.

For Google Chrome, the minimum hardware and software requirements include running the Windows 7 OS or a later version, the OS X Yosemite version 10.10 or later, and the 64-bit versions of either Ubuntu 14.04 or later, Debian 8 or later, OpenSUSE 13.3 or later, or Fedora Linux 24 or later. Unless you are running a version of the OS X operating system, you will also need to have an Intel Pentium 4 processor or later version (as long as it is SSE2 capable).

Installation

No installation is required. This product runs from the web browser from the URL https://adeeb10abbas.github.io/WhereToPoop/

Troubleshooting

In the event of an error with the website, it is recommended that network settings be checked to confirm an Internet connection and the website restarted by closing the web browser, re-opening the browser, and returning to the website.

About:

Our project, Where to Poop, was created by five Drexel College of Computing and Informatics students to help people find toilets when they need them. Since our tagline is, "Yelp for Toilets," we include ways to rate toilets and to find toilets based on your own preferences and needs. Toilets can easily be added to the map by filling out a simple form, and once your added toilet is ready, we'll alert you that it's on the map.
